# JobBoard - Web Development Project

## Overview
JobBoard is a web-based platform that enables employers to post job openings and job seekers to search and apply for jobs. This project is built using **React**, **Node.js**, and **MongoDB**. It focuses on providing a **responsive, user-friendly interface** with essential functionalities for both candidates and employers.

---

## Features

- **Home Page**: Welcome message and featured job listings.  
- **Job Listings Page**: View all available job openings with key details.  
- **Job Detail Page**: Detailed information for a specific job.  
- **Employer Dashboard**: Manage account and post job openings.  
- **Candidate Dashboard**: Manage profile and applications.  
- **Job Application Process**: Apply to jobs with resume upload.  
- **Search Functionality**: Search bar for finding specific jobs.  
- **Email Notifications**: Receive notifications for applications and updates.  
- **User Authentication & Security**: Secure login and data protection.  
- **Mobile Responsive**: Works seamlessly on desktops, tablets, and phones.

---

## Technologies Used

- **Frontend**: React, HTML, CSS, JavaScript  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Hosting**: Can be deployed on Netlify / Heroku / 000Webhost
