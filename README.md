# Quiz-App (Medical Remedies Quiz)

## Overview
Quiz-App is a **full-stack web application** that allows users to **create, take, and manage quizzes**. Users can input questions, multiple-choice answers, and correct answers to build a quiz. Other users can take the quiz and receive **immediate feedback** on their performance.  

This project demonstrates **backend and frontend integration**, user interaction, and a responsive interface. It is built with **Python (Flask)** and **HTML/CSS with Jinja2 templates**.  

---

## Features
- **Home Page:** Welcome message and options to create or take a quiz.  
- **Quiz Creation:** Users can create quizzes with multiple-choice questions.  
- **Quiz Taking:** Questions are displayed one at a time; users can select answers.  
- **Quiz Results:** Immediate feedback with final score and correct answers.  
- **Quiz Listing:** Browse all available quizzes.  
- **User Authentication:** Register and login for a personalized experience.  
- **Mobile Responsiveness:** Works well on desktop, tablet, and mobile.  

---

## Tech Stack
- **Backend:** Flask (Python)  
- **Frontend:** HTML, CSS, Jinja2 Templates  
- **Database:** Optional (can be added for storing quizzes and users)  
- **Version Control:** GitHub
